---

The Happy Next CLI wraps your existing AI coding tools (Claude Code, Codex, Gemini CLI) and adds mobile access, real-time sync, and orchestration.

## Requirements

- **Node.js 20** or later
- At least one AI coding tool installed:
  - [Claude Code](https://docs.anthropic.com/en/docs/claude-code/getting-started)
  - [Codex](https://github.com/openai/codex)
  - [Gemini CLI](https://github.com/google-gemini/gemini-cli)

## Install

```bash
npm install -g happy-next-cli
```

Verify the installation:

```bash
happy --version
# 0.2.3
```

## First run

Navigate to a project directory and run:

```bash
happy
```

This starts a Claude Code session (the default agent). On first run, Happy will:

1. Generate an encryption key pair stored at `~/.happy-next/access.key`
2. Register your machine with the Happy server
3. Start the **daemon** (background service) automatically
4. Display a QR code for mobile app pairing

<Warning>
**Keep your encryption key safe.** The file `~/.happy-next/access.key` is your identity and encryption key. If you lose it, you cannot decrypt your session history. Back it up.
</Warning>

## Data directory

Happy stores its data at `~/.happy-next/`:

| File | Purpose |
|------|---------|
| `access.key` | Your encryption private key |
| `settings.json` | CLI configuration |
| `daemon.state.json` | Daemon process info |
| `logs/` | Session and daemon logs |

## Update

```bash
npm update -g happy-next-cli
```

Or use the built-in command:

```bash
happy update
```

## Uninstall

```bash
npm uninstall -g happy-next-cli
```

To also remove local data:

```bash
rm -rf ~/.happy-next
```
